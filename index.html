Torchecker

Keeps a database of known tor exit nodes and after you launch tor run the script to check your tor exit node status

Download as .zip Download as .tar.gz View on GitHub

Main Use.

When your working with multiple dockers, images etc and you need to check if tor is enabled and working use this.
